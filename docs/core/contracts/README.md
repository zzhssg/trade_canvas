# Core Contracts（契约）

这里放“数据结构/协议”的最小真源，优先保证：
- 可落地（能写代码/能验收）
- 可演进（版本化；旧版本可回放/可兼容）
- 可对齐（前后端/回放/实盘共用同一口径）
