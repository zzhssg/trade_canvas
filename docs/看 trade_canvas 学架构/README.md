# 看 trade_canvas 学架构（专栏索引）

目标：让刚学过几天 C 语言的人，通关 30 关后，具备 10 年架构师的系统设计直觉。

写作方法：「白话发明法」——问题驱动、比喻先行、代码锚定、每关一个可复述的通用方法论。

## 使用方式

- 按编号顺序阅读，前一关的产物是后一关的输入。
- 每篇对应真实代码位置，读完可直接对照源码验证。
- 每关末尾有 5 道自测题，能用自己的话讲顺就算过关。
